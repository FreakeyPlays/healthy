## Run Locally

Clone the project

```bash
  git clone https://github.com/FreakeyPlays/healthy.git
```

Go to the project directory

```bash
  cd healthy/
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

## Run Tests

### Unit Tests

To run Unit tests, run the following command

```bash
  npm run test
```

### E2E Tests

To run e2e tests, run the following command

```bash
  npm run e2e
```
